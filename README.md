## ChatApl
chatApl is a simple mine java project which defines the interaction between Client and Server or Source to Destination in any messenging API.

## Installation

```bash
  $ git clone --depth=1 https://github.com/pz2100/chatApl
  $ cd chatApl
  $ ./run.sh
  ```
